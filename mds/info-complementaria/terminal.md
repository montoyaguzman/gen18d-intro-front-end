### Comandos Terminal

El **CMD** es la línea de comandos basada en MS-DOS para windows, mientras que la **terminal** es la línea de comandos de unix/lunix (kernel de mac os y algunas distribuciones linux).

Al escribir comandos en la CLI con los que ordenamos al sistema realizar acciones concretas.

**Nota:** Tambien se suele hacer referencia a ella como **CLI (Command Line Interface), línea de comandos, consola, bash**.



#### **Listado de comandos**

```
# imprime el path en el que se encuentra
pwd

# moverse entre carpetas
cd

# listar carpetas y archivos que hay en el directorio actual
ls

# crear carpetas
mkdir newFolder

# crear archivos
touch nameFile.txt

# eliminar carpetas
rmdir folderName

# eliminar archivos
rm fileName.txt

# comprobar conexión a internet
ping www.google.com

# limpiar la pantalla
clear
```
